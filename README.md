## sirpho components

#### 常用组件库

| 工具类     | 导入文件                 |
|---------|----------------------|
| 多列下拉组件  | ComboBox             |
| vxe筛选插件 | FilterExtend         |
| 页面容器    | PageContainer        |
| 查询容器    | QueryFilterContainer |
| 日志弹框    | StatusPop            |
| vxe容器   | VxeContainer         |

- [npm包地址](https://www.npmjs.com/package/@sirpho/components)


- [github地址](https://github.com/sirpho/components)

打包命令

```git
pnpm i
pnpm build
```

在项目中使用

```cmd
npm i @sirpho/components
```
